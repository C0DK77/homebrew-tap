# homebrew-tap

Tap Homebrew de C0DK77.

## Installer wptemps

```bash
brew tap C0DK77/tap
brew install --cask wptemps        # ajouter --no-quarantine pour eviter le clic-droit
```

wptemps : overlay des températures et infos matériel sur le bureau (Apple Silicon).
https://github.com/C0DK77/wptemps
